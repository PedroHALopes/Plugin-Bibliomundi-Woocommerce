== Description ==

For Woocommerce, with the aim of integrating the registered ebooks on our platform with your store. So that you can sell our ebooks in your store is very simple and does not need knowledge in programming.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bibliomundi-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the bibliomundi-woocommerce through the 'Plugins' screen in WordPress, make sure the Woocommerce plugin has installed before.
3. Use the Bibliomundi screen to configure the plugin

== Frequently Asked Questions ==

= Overview? =

After ebooks importing have finished, you can sell these ebooks on your store as virtual products.

= How to import ebooks? =

This is the time when you will import the registered ebooks on our platform to your store. Click on the Bibliomundi menu. You only need to enter the key and password that we sent to you, choose the desired environment and action and click Import. Attention. Import time will vary according to several factors, such as the speed of your server and the connection of your internet!
